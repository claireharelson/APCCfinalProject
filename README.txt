
* ABOUT *

Protein Structural Biology Visualization and Analysis Tool

	- Intended for the search of E. coli BL21 protein information


Source code can be obtained here:

	- http://bfx3.aap.jhu.edu/charels1/final.tar.gz


Utilizes tools published at:

	- https://biopython.org
		- Bio.PDB module used to parse PDB files


* REQUIREMENTS *

	- Functioning internet browser for using search tool


* TOOL USAGE *

1. Initial form can be found at http://bfx3.aap.jhu.edu/charels1/final/html/search.html

2. Enter the name of an Escherichia coli BL21 (D3) protein into the search bar and click submit.

3. Results of the query will then be displayed to the user.
